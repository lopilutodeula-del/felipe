# Security

Se você encontrar uma vulnerabilidade de segurança, por favor reporte privadamente para o mantenedor.

Recomenda-se enviar um e-mail para o endereço listado no perfil do mantenedor ou abrir uma issue marcada como "security" (se pública for adequada).

Processo recomendado:
- Descreva o impacto e passos para reproduzir (inclua versões afetadas).
- Se possível, inclua um patch ou mitigação temporária.
- Aguarde a resposta do mantenedor antes de divulgar publicamente.
