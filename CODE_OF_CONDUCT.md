# Código de Conduta

Este projeto adota o Código de Conduta do Contributor Covenant.
Para mais informações e detalhes sobre como reportar uma má conduta, consulte o arquivo upstream:

https://www.contributor-covenant.org/version/2/0/code_of_conduct/

Resumo:
- Seja respeitoso com os demais contribuidores.
- Não use linguagem ofensiva ou discriminatória.
- Reporte condutas inapropriadas abrindo uma issue privada ou contatando o mantenedor.
